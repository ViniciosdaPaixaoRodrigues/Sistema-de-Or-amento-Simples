# Sistema-de-Orçamento-Simples
Um programa feito em python que faz orçamento seguindo alguns parâmetros, separando os dados e podendo exportar o resultado para um arquivo .csv

# Como rodar:
1. Tenha certeza que tenha uma pasta selecionada para rodar o programa, ex: Visual Studio Code com uma pasta "Programa" selecionada como pasta de trabalho.
2. Só rodar o programa.
